# Math Quiz (python)
A simple command-line math quiz game written in Python.
The program generates random math problems and measures how fast you solve them.
It is a great beginner Python project to practice:
- Random number generation
- Loops
- Functions
- User input
- Time tracking
## Features
- Generates random math problems
- Supports multiple operators (+, −, ×, ÷) depending on configuration
- Tracks wrong answers
- Measures total time taken
- Simple terminal-based gameplay
##  Technologies Used
- Python 3
- random module
- time module

## How It Works
The program generates two random numbers.
A random mathematical operator is selected.
The user is asked to solve the problem.
If the answer is wrong, the program keeps asking until the correct answer is given.
The timer tracks how long it takes to solve all problems.

## How to Run
- Make sure Python is installed.
- Bash
- Copy code
- python Math_Quiz.py
- Then press Enter to start the quiz.


Example:


Press enter to start!

- Problem #1: 5 + 7 = ?12
- Problem #2: 8 * 3 = ?24
